Проект: "Обучение с учителем"

Структура:
1. Подготовка данных
2. Исследование задачи
3. Борьба с дисбаллансом
4. Тестирование моделей

Данный проект является первичным представлением автора на тему 
"Обучения с учителем" и будет иметь исправления по ходу времени. 
Все изменения будут зафиксированы, а информация о них будет 
донесена до участников проекта.
